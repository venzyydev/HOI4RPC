# HOI4RPC
HOI4RPC is a python script which will track your saves and import them in your discord presence status

TO RUN THIS SCRIPT PROPERLY YOU HAVE TO INSERT YOU CLIENT ID IN LINE 6
TO OBTAIN YOUR ID YOU MUST ACCESS THE DISCORD DEVELOPER PORTAL THROUGH YOUR BROWSER
NAME IT HOI4 AND ADD AN IMAGE (OPTIONAL) THEN COPY THE CLIENT ID AND IMPORT IT IN THE SCRIPT, LASTLY SAVE AND RUN HEARTS OF IRON 4 NORMALLY NO MODIFICATIONS REQUIRED INSIDE THE DOCUMENTS
